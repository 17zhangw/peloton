# N-Store 

DBMS designed for non-volatile memory (NVM).

## Dependencies

- **gcc (4.6+)** 
- **gmake (3.80+)**
- **flex (2.5.31+)** 
- **bison (1.875+)**
- **libreadline-dev** 
- **zlib**
- **perl (5.8+)**

## Setup
        
```
    ./bootstrap
    ./configure
    make
    make install
```

## Test

```
    make check
```
